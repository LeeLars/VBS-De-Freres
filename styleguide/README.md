# VBS De Frères Styleguide

**⚠️ INTERNAL ONLY - NOT FOR PUBLIC ACCESS**

Deze styleguide is alleen bedoeld voor intern gebruik door het ontwikkel- en designteam.

## Toegang

Open `index.html` lokaal in je browser om de styleguide te bekijken.

**Belangrijk:** Deze folder staat NIET in de `web/` directory en wordt dus niet gepubliceerd naar GitHub Pages.

## Inhoud

- **Kleurenpalet**: Officiële kleuren (#a8cf8c, #fffffb, #000)
- **Typografie**: ABeeZee font specificaties en groottes
- **Buttons**: 3 button stijlen met animaties
- **Spacing**: Consistente padding, margin en gap waarden
- **Layout**: Container widths, grid gaps, border radius

## Voor Designers

Gebruik deze styleguide als referentie bij het ontwerpen van nieuwe pagina's of componenten voor VBS De Frères.

Alle specificaties in deze guide moeten strikt worden gevolgd voor consistente branding.
